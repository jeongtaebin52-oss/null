# docs 폴더 목차

> **기준**: PROJECT.md = 스펙 원본. **완성**을 위해 따라 할 문서 = **실행_완성_실행_리스트.md**.  
> **기능·실행 체크는 실행_완성_실행_리스트.md 단일 소스.** 갭 정리·완성도·현재 작업 문서의 체크 항목은 모두 그 파일에 통합됨. 아래는 **지금 docs에 실제로 있는 파일만** 나열함.

## 1. 문서 구분

| 구분 | 용도 |
|------|------|
| **PROJECT.md** | 스펙·정의·와이어프레임·시나리오 **원본**. 완성 기준. |
| **실행_** | 따라 할 작업 — **완성까지** 체크리스트. |
