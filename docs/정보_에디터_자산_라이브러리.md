## 0. 표기 규칙

- `[기존구현]` = 이미 `PRESET_GROUPS` 등에 존재하는 항목
- `[구현완료]` = 자산 프리셋 삽입/구조/디자인 적용 완료
- `[기능예정]` = 프로토타입/데이터 바인딩 등 동작 추가 예정
- `[미구현]` = 신규 기획 또는 백로그
