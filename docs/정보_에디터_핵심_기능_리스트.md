# 에디터 핵심 기능 — 최종 리스트

> **상태**: 정리 완료 (정보 문서)
> **목적**: “앱 기능”이 아닌 **에디터 자체 기능**만 정리한 최종 리스트. 실제 제작자 요구 기준으로 **우선순위(사용자 만족도) + 전체 기능**을 통합.

---

## 0. 제외 항목 (요청 반영)

- 화이트보드/브레인스토밍 보드
- 프레젠테이션/슬라이드 제작 모드
- 파일/페이지/컴포넌트 권한·공유·승인 체계
- 조직/팀 관리(사용자/권한/감사 로그/정책)
- 디자인 시스템 릴리스/버전 관리(배포·회수)
- 라이브러리 사용 현황/의존성 분석
- AI 보조 기능(레이아웃/텍스트/스타일 자동화)

---

## 1. 최우선 5 (사용자 만족도 기준)

1. [완료] 버전 관리/스냅샷/복원 (작업 안정감, 실무 만족도 1순위)
2. [완료] 컴포넌트/인스턴스/오버라이드 (재사용·유지보수 효율 급상승)
3. [완료] 리스폰시브 브레이크포인트 + 제약 고도화 (실사용/배포 직결)
4. [완료] 데이터 바인딩 + 반복 리스트 (실제 서비스처럼 보이게 만드는 핵심)
5. [완료] 자산 라이브러리 관리(이미지/폰트/아이콘/대량 교체)
