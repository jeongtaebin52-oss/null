## 실행 순서 (묶음)

### Phase 1 — 빠른 승리 (속성·레이어·버전 UI) ✅ 완료

**목표**: UI만 추가하거나 API 연동만으로 끝나는 것. 기존 로직 건드리지 않음.

| # | 항목 | 작업 내용 | 완료 |
|---|------|-----------|------|
| F4 | 블렌드 모드 드롭다운 | 우측 패널 "채우기·테두리·효과" 섹션에 blendMode &lt;select&gt; 추가. scene.BlendMode 값으로 선택·updateNode 반영. | ✅ |
| N5 | 버전 히스토리 UI | GET /api/pages/[pageId]/versions 호출해 목록 표시. "복구" 시 POST version/restore 호출 후 doc 갱신. (메뉴 또는 우측 상단) | ✅ |

**산출**: 블렌드 선택 가능, 버전 목록·복구 가능.
