## 5. 플러그인 — 구현 가능성 (요약)

| 항목 | 내용 |
|------|------|
| **난이도** | 높음. Figma 플러그인은 샌드박스·API·UI 런타임이 따로 있음. |
| **필요 요소** | (1) 플러그인 런타임(iframe/worker 등), (2) Doc/선택/노드 노출 API, (3) UI 위젯 규약, (4) 보안·격리 정책. |

## 7. Figma 에디터 기준 완성도 (로드맵 아님, Figma 대비만)

**기준**: Figma 에디터가 가진 기능 전체(디자인·프로토타입·개발/내보내기). 협업·플러그인·팀 라이브러리·브랜치는 제외.

| Figma 기준 범위 | 완성도 | 있는 것 / 없는 것 |
|-----------------|--------|-------------------|
| **Figma 에디터 전체 (협업·플러그인 제외)** | **약 62~68%** | 아래 구간 합산. |
| ─ 도구·레이아웃·스타일 | 높음 | 선택·프레임·도형·펜·텍스트·이미지·비디오·Boolean·마스크·오토 레이아웃·제약·그리드·채우기·테두리·다중 효과·블렌드 모드·스타일 토큰·변수. |
| ─ 컴포넌트·인스턴스 | 높음 | 생성·인스턴스·변형(Variants)·오버라이드·스왑. |
| ─ 프로토타입 | 높음 | 이동·뒤로·오버레이·scrollTo·전환·setVariant·setVariable·조건(condition). 부족: 스크롤 트리거(위치→액션). |
| ─ 개발·내보내기 | 높음 | CSS·스펙·자산 추출·내보내기(페이지/선택·배율). |
| ─ 타이포·벡터·미디어 | 중간 | 기본 타이포·크롭·밝기·대비·path·Boolean. 부족: OpenType/변수 폰트·Vector network(세그먼트별 다중 fill). |
| ─ 성능·규모 | 중간 | 뷰포트 컬링·LOD. 부족: 수천 노드 스트레스 검증. |
| ─ 접근성 | 낮음 | 키보드·ARIA 일부. 부족: 전면 포커스·스크린 리더. |
| **협업 (Figma 포함)** | **0%** | 정식 출시 이후 예정. |

**한 줄**: Figma 에디터만 놓고 보면 **지금 약 65% 전후**. 도구·레이아웃·스타일·컴포넌트·프로토타입·개발/내보내기는 대부분 있고, OpenType·Vector network·스크롤 트리거·대용량 검증·접근성이 갭.

## 8. Figma 에디터 90% 이상 — 필수 기능 리스트 (사소한 디테일 포함)

**원칙**: 로드맵이 아니라 **Figma 기준**으로, 90%+ 달성을 위해 **꼭 챙겨야 할 것 전부** 나열. 사소한 디테일까지 포함.

**완료 시 목표 수치**: 이 리스트(A~N)를 **전부** 완료하면, Figma 에디터(협업·플러그인·팀 라이브러리 제외) 기준 **약 92~95%**. (Vector network(I5)는 장기/선택으로 두고 제외해도 92% 이상 유지.)
